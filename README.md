# Test MediaHaus

## Para correr el programa:
Clonar el repositorio y luego en la carpeta del proyecto ejecutar:

### `yarn install`
### `yarn start`
or
### `npm install`
### `npm start`

Es necesario tener instalado NodeJS

## Abrir 
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
